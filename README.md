# Global Semiconductor Industry Analysis 2025

## Overview
Analysis of top 153 semiconductor companies by market cap, 
country distribution, and stock price.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI

## Key Findings
- USA dominates with 71% of total market cap
- NVIDIA alone holds 40% share among top companies
- Netherlands has highest average stock price
